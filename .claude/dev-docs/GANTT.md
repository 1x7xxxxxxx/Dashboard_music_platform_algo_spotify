## How to update this file

1. **When a brick changes status** → update BRICKS.md Summary Table AND the matching line in the Mermaid gantt above.
2. **Run the stats regeneration** : `python3 tools/generate-dev-docs.py --gantt` — updates the `AUTO:GANTT_STATS_*` block only (does not touch the Mermaid body).
3. **Commit both** BRICKS.md + GANTT.md together to keep them in sync.
